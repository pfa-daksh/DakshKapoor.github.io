ECommerce business analysis in Python

Online purchasing has become increasingly popular since 2020, with customers preferring convenience and ease of shopping from home or workplace. However, understanding the intentions and behavioral patterns of online shoppers is challenging. This study classifies customers using machine learning techniques based on browsing parameters and dimensions. The analysis used secondary data from Kaggle Machine Learning Repository, with bagging and boosting algorithms used to predict purchasing intentions. The study found that May had the highest revenue, repeat visits, and special days. Gradient Boosting algorithm provided the highest accuracy in predicting consumer behavior, while Light GBM and Random Forest achieved the highest accuracy in up and down sampling.
